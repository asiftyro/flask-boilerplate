# flask-boilerplate


### Create virtual env

```
python3 -m venv venv
```

### Activate virtual env

```
source vemv/bin/activate
```

### Install packages

```
pip install -r requirements.txt
```

### Run app in dev mode

```
export FLASK_APP=flaskr
export FLASK_ENV=development
flask run
```


--------------------------


### Deactivate venv
```
deactivate
```
